## Intro
<!-- type: presenter -->
What if a software company ran almost entirely on AI agents, with humans steering for less than an hour a day? That's B4arena. An agentic company where AI agents write the code, track the work, review each other's pull requests, and ship to production.

## How It Works
<!-- type: presenter -->
B4arena has three nested feedback loops. At the top, humans evolve the specs and agent identities over weeks. In the daily loop, agents execute tasks while humans make quick steering decisions. And in the inner loop, each agent reasons and acts within boundaries defined by its SOUL file. Think of it as a self-driving software team, where you're the navigator, not the driver.

## The Stack
<!-- type: presenter -->
Every agent has a SOUL file that defines its personality and rules. They coordinate through Beads, a Dolt-powered issue tracker that survives conversation resets. Code review happens automatically through the Four Eyes Protocol, where at least two agents must approve every change. And the whole system is designed so that no single agent can make a dangerous decision alone.
